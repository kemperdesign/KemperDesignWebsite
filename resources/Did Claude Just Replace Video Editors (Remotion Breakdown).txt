Chapter 1: What We're Covering

Just like we've seen vibe coding and vibe automation, we now have vibe editing. This is powerful for anyone wanting to build businesses and make money online.

This introduces using Claude Code with Remotion to generate videos quickly, almost like having a professional editor.

Chapter 2: Prompt-to-Video with Remotion + Claude

You can now:

Create AI avatars
Add motion graphics
Generate full explainer videos
Stitch clips into cinematic sequences

All by prompting.

Remotion works as a code-based video editor, allowing:

Positioning (left/right, overlays)
Layering assets
Combining clips, audio, and visuals

Claude can:

Access project assets
Insert them into scenes
Automatically structure videos
Chapter 3: Installing the Remotion Skill

The process:

Open Claude Desktop
Create a new project folder
Install the Remotion skill (via command)

Then prompt:

Fetch branding (colors, logos, etc.)
Generate a video
Build the Remotion project

Claude will:

Install dependencies
Create the project
Generate video structure
