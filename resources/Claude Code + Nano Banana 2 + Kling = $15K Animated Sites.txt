Chapter 2: Generating Websites Easily

The reason I want to make this is to show how easy it is to generate websites like this. A few years ago, this would cost $5,000–$10,000 per site. Now you can do it in under 10 minutes for a few dollars in tokens.

The flow is simple:

Write bullet points into Claude Code
Use a design skill
Generate an animated asset
Integrate it and push live

There’s an open repo skill that applies high-end design principles automatically, making it easy to generate high-quality websites from a single prompt.

Chapter 3: Using Claude Code

You can one-shot entire websites using Claude Code.

The process:

Load the design skill
Give a prompt like:
“Design a high-end website about interior design”
Let it generate the full site

For animation:

Use a model like Cling 3.0
Generate short 3D video assets
Combine them into the site

This typically costs only a few dollars total.

Chapter 4: Setting Up the Design

Steps:

Run Claude Code inside an IDE (like Anti-Gravity)
Load the design skill repo
Prompt it to generate a full website

Then generate an animation:

3D scene
White background
High quality
16:9 format
Short duration

Download the animation and insert it into the website as a hero background.

Chapter 5: Enhancing the Animation

Next, improve the site:

Add scroll-based animation
Generate a second asset (e.g., exploding house view)
Insert it below the hero section

Prompt example:
“Create a scroll animation tied to this video showing design features as the user scrolls.”

You can:

Adjust readability (text contrast, size)
Improve animation quality
Generate multiple variations
